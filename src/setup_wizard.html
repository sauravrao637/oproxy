3 Configure your client
System-wide (GNOME)
  1. Settings → Network → Network Proxy → Manual
  2. HTTP Proxy: 127.0.0.1   Port: 8080
  3. HTTPS Proxy: same host and port.
  4. Apply.
Environment variables (terminal / CLI tools)
export http_proxy="http://127.0.0.1:8080"
export https_proxy="http://127.0.0.1:8080"
export no_proxy="localhost,127.0.0.1"
# Add to ~/.bashrc or ~/.profile to persist
Trust the CA certificate
# Download
curl -o oproxy-ca.crt http://127.0.0.1:8080/admin/ca

# Debian / Ubuntu / Mint
sudo cp oproxy-ca.crt /usr/local/share/ca-certificates/oproxy.crt
sudo update-ca-certificates

# Fedora / RHEL / CentOS
sudo cp oproxy-ca.crt /etc/pki/ca-trust/source/anchors/
sudo update-ca-trust extract

# Arch
sudo trust anchor --store oproxy-ca.crt
System proxy
  1. System Settings → Network → select interface → Details → Proxies
  2. Enable Web Proxy (HTTP) and Secure Web Proxy (HTTPS)
  3. Host: 127.0.0.1   Port: 8080
  4. OK → Apply
Terminal equivalent
networksetup -setwebproxy Wi-Fi 127.0.0.1 8080
networksetup -setsecurewebproxy Wi-Fi 127.0.0.1 8080
Trust CA certificate
  1. Download the cert above, double-click → opens Keychain Access
  2. Find oproxy CA under System → double-click → Trust → Always Trust

sudo security add-trusted-cert -d -r trustRoot \
  -k /Library/Keychains/System.keychain oproxy-ca.crt

curl

     --cacert oproxy-ca.crt \
     https://api.example.com/endpoint
wget

     --ca-certificate=oproxy-ca.crt \
     https://api.example.com/endpoint
Python (requests)
import requests
proxies = {"http": "http://127.0.0.1:8080",
           "https": "http://127.0.0.1:8080"}
r = requests.get("https://api.example.com/", proxies=proxies,
                 verify="oproxy-ca.crt")
Node.js (undici / fetch)
const { ProxyAgent } = require('undici');
const agent = new ProxyAgent('http://127.0.0.1:8080');
process.env.NODE_EXTRA_CA_CERTS = 'oproxy-ca.crt';
fetch('https://api.example.com/', { dispatcher: agent });
Go (net/http)
proxyURL, _ := url.Parse("http://127.0.0.1:8080")
caCert, _ := os.ReadFile("oproxy-ca.crt")
pool := x509.NewCertPool(); pool.AppendCertsFromPEM(caCert)
client := &http.Client{Transport: &http.Transport{
    Proxy:           http.ProxyURL(proxyURL),
    TLSClientConfig: &tls.Config{RootCAs: pool},
}}
