Root CA Certificate

Install the oproxy root CA certificate in your system or browser trust store to enable HTTPS interception. After installing, set mitm.enabled: true in configs/default.yaml and restart.

macOS: Double-click the cert → Keychain Access → right-click → Get Info → Trust → Always Trust
Firefox: about:preferences → Privacy → View Certificates → Authorities → Import
Chrome / Edge: Settings → Privacy → Manage Certificates → Trusted Root Authorities → Import
MITM Status
Mode—

When MITM is on, CONNECT tunnels are intercepted: the proxy terminates TLS using a per-domain certificate signed by the root CA, decrypts and inspects each request, then re-encrypts to the upstream. Install the root CA in your system or browser trust store to avoid certificate warnings.
